# Imagine Engine #

**Imagine Engine** - Open Source 3D game engine written in C++.

Supported platforms: Windows, Linux, Mac OSX

# Dependencies #

-C++11

-OpenGL

-SDL2, SDL2_image, SDL2_ttf

-GLEW

-GLU
